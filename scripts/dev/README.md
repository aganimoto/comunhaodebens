# Scripts de Desenvolvimento

Scripts utilitários para setup e seed do banco de dados em ambiente de desenvolvimento.

## Scripts

| Arquivo | Função |
|---|---|
| `seed_sheets.py` | Povoa a planilha Google Sheets com dados de exemplo |
| `setup.sh` | Script de setup inicial para ambiente Linux/Mac |

## Uso

```bash
# A partir da raiz do projeto
python scripts/dev/seed_sheets.py

# Linux/Mac
bash scripts/dev/setup.sh
